# Diventa Website

Riproduzione del sito web di **[Diventa S.r.l.](https://diventa.it)** realizzata come esercizio pratico di HTML e CSS.

## Descrizione

Questo progetto è un clone statico del sito istituzionale di Diventa, una società di consulenza che integra strategia, behavioural science e design della comunicazione. L'obiettivo dell'esercizio è ricreare fedelmente il layout e lo stile del progetto Figma sul quale si basa il sito, utilizzando esclusivamente HTML e CSS, senza framework o librerie JavaScript.

## Struttura del progetto

```
Diventa-website/
├── index.html           # Homepage
├── expertise.html       # Aree di competenza
├── solutions.html       # Soluzioni
├── case-studies.html    # Case Studies
├── contacts.html        # Contatti
├── css/
│   └── style.css        # Foglio di stile principale
└── img/                 # Immagini e asset grafici
```

## Pagine

| Pagina | File | Descrizione |
|---|---|---|
| Homepage | `index.html` | Hero, presentazione, servizi offerti e partner |
| Aree di competenza | `expertise.html` | Dettaglio delle aree di intervento |
| Soluzioni | `solutions.html` | Panoramica delle soluzioni proposte |
| Case Studies | `case-studies.html` | Progetti e clienti |
| Contatti | `contacts.html` | Form e informazioni di contatto |

## Tecnologie utilizzate

- **HTML5** — struttura semantica delle pagine
- **CSS3** — layout, tipografia, animazioni (marquee) e responsive design
- **Google Fonts** — font [Geist](https://fonts.google.com/specimen/Geist)

## Funzionalità implementate

- Layout responsive con breakpoint mobile/desktop
- Navigazione multi-pagina
- Sezione hero con testo promozionale
- Griglia di card per i servizi (Strategy, Learning, Behavioural Change, Digital Solutions)
- Striscia animata con loghi dei partner (marquee)
- Header con logo e menu desktop
- Footer con informazioni aziendali (P.IVA, indirizzo, contatti)
- Immagini responsive con `<picture>` e `srcset`

## Come visualizzare il progetto

1. Clona la repository:
   ```bash
   git clone https://github.com/pavandesiree/Diventa-website.git
   ```
2. Apri il file `index.html` nel tuo browser.

Non è richiesta nessuna installazione o dipendenza esterna.
